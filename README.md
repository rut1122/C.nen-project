# Order Management System - .NET WinForms

מערכת לניהול מוצרים, לקוחות והזמנות שנבנתה כפרויקט גמר בקורס .NET. המערכת מדגימה עבודה בארכיטקטורת שכבות, ניהול נתונים ב-XML וממשק משתמש מודרני ב-WinForms.

## 🚀 תכונות עיקריות
* **ניהול קטלוג מוצרים:** הצגה, הוספה, עדכון וסינון מתקדם של מוצרים לפי קטגוריות וחיפוש חופשי.
* **ניהול עגלת קניות:** הוספת מוצרים לסל, עדכון כמויות וחישוב מחיר סופי בזמן אמת.
* **מועדון לקוחות:** זיהוי לקוחות קיימים והחלת הנחות מועדון/מבצעים רלוונטיים.
* **ניהול מבצעים (Sales):** הגדרת מבצעים מוגבלי זמן או מיועדים לחברי מועדון בלבד.
* **ארכיטקטורת שכבות (N-Tier):** הפרדה מלאה בין שכבת התצוגה, הלוגיקה והנתונים.

## 🏗 ארכיטקטורה
הפרויקט מחולק ל-4 שכבות מרכזיות:
1. **UI (User Interface):** ממשק המשתמש שנבנה ב-WinForms על גבי .NET 6/8.
2. **BL (Business Logic):** שכבת הלוגיקה העסקית המנהלת את כל החישובים והסינונים.
3. **BO (Business Objects):** ישויות הנתונים המשותפות לכל השכבות (Classes & Enums).
4. **DAL (Data Access Layer):** שכבת הגישה לנתונים, המבצעת Serialization ו-Deserialization לקבצי XML.

## 🛠 טכנולוגיות
* **Language:** C#
* **Framework:** .NET 6.0 / .NET 8.0
* **Data Storage:** XML Files
* **Tools:** LINQ, XML Serialization, Git

## 📂 מבנה התיקיות

* `BL/` - מימוש הלוגיקה העסקית והפונקציות המרכזיות.
